电竞微博分析

发表于 2020-05-17

方法

1、数据获取：使用公开的爬虫（https://github.com/dataabc/weiboSpider）爬取“IG电子竞技俱乐部”新浪账号发布的大部分原创微博，及FPX战队账号的微博数据以对比。
2、数据分析：
1）基于以上数据分析微博转评数随时间的变化。考虑到微博点赞功能是在18年左右上线，而IG微博从12年已经开始，故未将点赞数纳入分析项；
2）基于python的jieba及wordcloud包进行词频统计并生成词云；
3）基于微博内容和所带tag进行细节分析。

注意事项

  • 在安装包时需通过python3 -m pip install *** 来安装到3.0版本
  • 微博cookie及user id的获取见上面链接
  • 通过更改json文件中的内容并运行python WeiboSpider.py来实现数据爬取
